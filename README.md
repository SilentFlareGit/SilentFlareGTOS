# SilentFlareGToS

**SilentFlareGToS** means **SilentFlare General Terms of Service**.

Single-page Markdown policy document for the SilentFlare personal blog.

## Structure

```text
GToS.md
README.md
docs/
  _config.yml
  GToS.md
  index.md
```

GitHub Pages is configured to publish from `main` / `docs`.

The public webpage is `docs/index.md`, which renders `docs/GToS.md` as one page. The root `GToS.md` is kept as a repository-readable copy.

## Editing

For the public webpage, edit `docs/GToS.md`.

For the root Markdown copy, keep `GToS.md` synchronized when practical.

Use the exact short name `SilentFlareGToS` and the exact full name `SilentFlare General Terms of Service`.

## Notice

This is a personal blog policy draft, not legal advice.
