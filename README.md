# SilentFlareGTOS

Static HTML version of the SilentFlare Global Terms of Service.

## Files

- `index.html` — standalone Terms of Service / Acceptable Use Policy page.

## Deploy

This repository can be deployed directly with GitHub Pages, Cloudflare Pages, Vercel, Netlify, or any static file server. No build step is required.
